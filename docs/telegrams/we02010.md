## 概要
特別警報に対応した気象警報・注意報を配信

**資料名** <br/>
 特別警報・警報・注意報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VPWW53
 
**/Report/Control/Title** <br/>
 特別警報・警報・注意報

**WebSocket圧縮形式** <br/>
 gzip
